# 한글 터치 쓰기 (알파 프로토타입)

## 실행 방법
- `index.html`을 브라우저에서 직접 열면 됩니다. (모바일/아이패드도 가능)
- 캔버스의 **주황색 시작점**에서 시작해 **점선 경로**를 손가락/마우스로 따라 그리세요.
- 각 획이 통과되면 **초록색**으로 채색되고, 모든 획을 통과하면 **보상 창**이 뜹니다.
- `다음`을 누르면 단어 단계가 `가 → 강 → 강아 → 강아지`로 진행됩니다. (현재는 시각적 표시 중심)

## 폴더 구조
```
assets/
data/
src/
index.html
style.css
```

## 참고
- 인식은 경량 템플릿 매칭(리샘플+평균거리) 기반으로 동작합니다.
- 난이도(관용치)는 상단 컨트롤에서 변경할 수 있습니다.
- 이 버전은 **'가'** 글자의 두 획(ㄱ, ㅏ)만 포함합니다. 파일 `data/ga.json`을 참고하여 다른 글자를 추가할 수 있습니다.
